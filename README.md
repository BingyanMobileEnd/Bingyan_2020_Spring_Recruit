## 实现一个浏览器

Feature

支持浏览网页

支持基本操作如：前进、后退、刷新


pluses

支持多窗口页面

支持 通过左右滑 进行前进后退 操作

有搜索历史、浏览历史管理

支持多种设置：比如 调整网页字体大小、切换搜索引擎、切换白天黑夜两种主题模式、是否无图浏览、查看保存网页图片等


key words

Android - WebView、 RecyclerView、GestureListener
